# Changelog

User-facing release notes for AteraSnipeSync are documented here beginning with version 1.0.1.

## [1.0.1] - 2026-07-23

### Fixed

- Preview runs no longer replace the **Latest run** timestamp or asset counts on the Dashboard. Preview results still appear under **Current activity** and in the manual log.
- The all-users Start Menu shortcut now consistently displays the AteraSnipeSync product icon after installation or upgrade.
